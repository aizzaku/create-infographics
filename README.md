# Infographic Creator Skill

> Creates stunning, information-dense visual infographics that look hand-crafted by a senior designer — not generated by AI. 

This skill empowers your agent to create production-grade infographics in interactive HTML, static retina PNG, and PDF formats from any data or brief. 

## 🛠️ Prerequisites & Dependencies

To use the export and rendering pipeline (PNG/PDF creation) effectively, ensure you have:

```bash
pip install playwright>=1.40.0 --break-system-packages
playwright install chromium --with-deps
```

*Note: The primary artifact is the self-contained HTML file which does not strictly require Playwright if you only need the web version.*

## 📦 Installation

To install this skill for your agent, you simply need to merge this repository into your agent's skills directory.

### Quick Install (via npx)
The most common and easiest way to install this skill directly into your agent's environment is using `npx`:

```bash
npx skills add https://github.com/aizzaku/create-infographics
```

### Claude Code CLI Integration
If you are using Anthropic's [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) CLI, you can directly run the skill command in your terminal. This allows Claude to leverage the `SKILL.md` instructions and execute the `export.py` scripts directly in your local environment.
```bash
claude "Create an infographic of our Q4 results using the data in Q4.csv"
```
*Note: Make sure the `infographics-skill` folder is accessible in your current working directory or installed via `npx` so Claude Code can automatically discover its context.*

### Claude.ai Projects (Web)
You can also use this skill directly in the web browser via Claude.ai Projects:
1. Create a new Project in Claude.ai.
2. Upload the `SKILL.md` file to the Project Knowledge.
3. Chat with Claude! It will read the instructions and generate the HTML code for your infographic. 
*(Note: Since the web interface cannot run Python scripts, you will not be able to use the `export.py` PDF/PNG generation tool in the browser. However, you can save the HTML file locally and run the export script yourself).*


### Manual Installation
1. Clone or download this repository.
2. Copy the entire `create-infographics` folder into your agent's `skills/` or `plugins/` directory.
3. Reload your agent or restart your agent environment. The agent will automatically read `SKILL.md` and understand how to use the skill.

## ⚡ When to Use This Skill

Trigger this skill on any of these signals:
- "create an infographic" / "make a visual summary" / "visualize this data"
- "one-pager" / "visual report" / "shareable graphic"
- Converting tables, comparisons, timelines, feature lists, or stats into visuals
- "poster" or "visual" combined with data or information content
- Any request where the user wants information presented as a designed image rather than text

**Do NOT use for**: pure art/illustration, slide decks, charts only, or standard documents.

## ✨ Features

- **Multi-Format Output**: Generates self-contained interactive HTML, 2x Retina PNGs, and PDFs.
- **Smart Data Intake**: Analyzes structured data, unstructured text, or mixed content to determine the best infographic archetype (e.g., timeline, product-launch, token-economics, comparison, etc).
- **Brand-Aware Design**: Automatically extracts dominant colors from logos and adapts the design system to match brand guidelines.
- **Rich Component Library**: Built-in support for hero blocks, KPI strips, comparison columns, roadmap timelines, data tables, and dynamic charts (using pure CSS or inlined Chart.js).
- **Offline / Self-Contained**: Can function offline with inlined Phosphor Icons and CSS, or leverage CDNs for smaller output files.
- **Live Editable**: Offers a JSON-based Live Editor block for incredibly easy post-generation tweaks without rewriting code.

## 🚀 How it Works (Execution Checklist)

The skill follows a strict six-phase execution pipeline to ensure high-quality output every time:

1. **Phase 0: Visual Reference** — Scans existing `examples/` to calibrate to a high-quality stylistic anchor.
2. **Phase 1: Intake** — Classifies the content type (e.g., listicle, ecosystem map) and brand config (logos, hex codes).
3. **Phase 2: Design Selection** — Selects an unconventional but harmonious theme, typography scale, and an 8px layout grid.
4. **Phase 3 & 4: Build & Polish** — Assembles components in a single self-contained HTML file with inline CSS, Phosphor icons, and subtle CSS animations.
5. **Phase 5: Export** — Generates PNG and PDF formats using Playwright (`scripts/export.py`).
6. **Phase 6: QA** — Verifies visual hierarchy, contrast, technical loaded assets, and multi-format success.
7. **Phase 7: Live Editor / Revisions** — Presents the user with a JSON block to easily tweak text and colors without requiring prompts or code edits.

## 🌍 Localization Support
The skill automatically adjusts styling, typography, and text-direction based on the requested language (e.g., `rtl` layouts for Arabic/Hebrew with Noto Sans Arabic, and zero letter-spacing adjustments for CJK languages).
